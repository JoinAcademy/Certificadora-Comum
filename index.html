A JOIN Academy tem como objetivo ensinar sobre os conceitos básicos de banco de dados para todos os interessados, abrangindo o maior número de pessoas possíveis com um ensino de qualidade.

Queremos que, a partir de exemplos práticos, seja possível exercitar o que foi visto nas vídeo aulas. Relacionando cada conteúdo com competências profissionais essenciais.

Nossos conteúdos foram dividos em 4 diferentes módulos:
1. Fundamentos de Banco de Dados e SQL;
2. Modelagem e Normalização;
3. Consultas SQL Intermediárias;
4. Validação e Teste de Resultados
